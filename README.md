# Boocave

나만의 독서 창고 북케이브

Boocave는 읽은 책을 검색하고 독후감을 작성하여 개인 책장을 관리할 수 있는 웹 애플리케이션!
도서를 검색하고, 별점 평가와 함께 독후감을 기록할 수 있습니다.

## 주요 기능

- 도서 검색 (Google Books API 연동)
- Google 소셜 로그인
- 별점 평가 (1-5점)
- 독후감 작성 및 저장
- 개인 책장 관리
- 독후감 상세보기 및 삭제

## 기술 스택

- Next.js 15
- Next.js API Routes
- NextAuth.js
- React 19
- TypeScript
- TailwindCSS
- Google OAuth
- Prisma ORM
- Supabase (PostgreSQL)
